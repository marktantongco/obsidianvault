# Plugin Setup Guide

## Required Plugins

Install these plugins through Obsidian's Community Plugin browser:

### How to Install
1. Open Obsidian Settings (Ctrl+,)
2. Go to **Community Plugins**
3. Click **Browse**
4. Search for each plugin name
5. Click **Install** then **Enable**

### Plugin List

#### 1. Dataview
- **Purpose**: Query and display data from your vault
- **Search**: "Dataview"
- **Author**: Michael Brenan
- **Use case**: Create dynamic tables, lists, and queries

#### 2. Calendar
- **Purpose**: Calendar view for daily notes
- **Search**: "Calendar"
- **Author**: Liam Cain
- **Use case**: Navigate daily notes visually

#### 3. Templater
- **Purpose**: Advanced template system
- **Search**: "Templater"
- **Author**: SilentVoid13
- **Use case**: Create complex templates with variables

#### 4. Kanban
- **Purpose**: Kanban boards for task management
- **Search**: "Kanban"
- **Author**: Matt Peerce
- **Use case**: Visual task management

#### 5. Tasks
- **Purpose**: Task tracking across vault
- **Search**: "Tasks"
- **Author**: Martin Schenck
- **Use case**: Track tasks with due dates, recurring tasks

#### 6. Excalidraw
- **Purpose**: Drawing and diagramming
- **Search**: "Excalidraw"
- **Author**: Zsolt Viczian
- **Use case**: Create diagrams, flowcharts, sketches

## Optional Plugins

These are also recommended:

- **Mind Map** — Visual thinking maps
- **Admonition** — Callout boxes
- **Style Settings** — Theme customization
- **Hider** — Clean UI mode

## Configuration

After installing, configure each plugin in Settings:

### Dataview Settings
- Enable JavaScript Queries: ON
- Enable Inline Queries: ON

### Calendar Settings
- Show Week Number: ON
- Start Week on Monday: ON (optional)

### Templater Settings
- Template Folder Location: templates
- Trigger Templater on new file creation: ON

### Tasks Settings
- Default Priority: Normal
- Auto-suggest in edit mode: ON

## Hotkeys to Set

After installing, set these hotkeys in Settings > Hotkeys:

- **Dataview: Refresh** → Ctrl+Shift+R
- **Calendar: Open** → Ctrl+Shift+C
- **Templater: Insert Template** → Ctrl+Shift+T (may already be set)
- **Kanban: Create new board** → Ctrl+Shift+K
- **Tasks: Create new task** → Ctrl+Shift+Alt+T

## Verification

After installing all plugins, verify they work:

1. Open Command Palette (Ctrl+Shift+P)
2. Type "Dataview" — should show Dataview commands
3. Type "Calendar" — should show Calendar commands
4. Type "Templater" — should show Templater commands

If commands appear, plugins are installed correctly.
